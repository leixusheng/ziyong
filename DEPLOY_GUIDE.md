## 🇨🇳 方案一：Gitee Pages（推荐）

### 步骤1：注册 Gitee 账号
访问 https://gitee.com 注册并登录

### 步骤2：创建新仓库
1. 点击右上角 "+" → "新建仓库"
2. 仓库名称：`family-meal-planner`（或任意名称）
3. 是否开源：选择 **公开**
4. 点击 "创建"

### 步骤3：上传文件
有两种方式：

**方式A：通过网页上传（适合少量文件）**
1. 进入仓库，点击 "上传文件"
2. 将 `dist` 文件夹内的所有文件拖拽上传
3. 提交更改

**方式B：使用 Git 命令（推荐）**
```bash
cd e:\自定义项目

# 初始化git（如果还没有）
git init
git add dist
git commit -m "Initial commit"

# 关联Gitee仓库（替换为你的仓库地址）
git remote add origin https://gitee.com/你的用户名/family-meal-planner.git

# 推送
git push -u origin master
```

### 步骤4：开启 Gitee Pages
1. 进入仓库页面
2. 点击顶部导航栏的 **"服务"** → **"Gitee Pages"**
3. 部署分支：选择 `master` 或 `main`
4. 部署目录：留空（或填写 `dist` 如果你推送的是整个项目）
5. 强制使用 HTTPS：勾选
6. 点击 **"启动"**

### 步骤5：访问应用
等待1-2分钟部署完成后，你会获得一个网址，例如：
```
https://你的用户名.gitee.io/family-meal-planner/
```

在手机浏览器中打开这个网址即可使用！

## 🇨🇳 方案二：腾讯云静态网站托管

### 步骤1：注册腾讯云
访问 https://cloud.tencent.com 注册账号

### 步骤2：开通 COS 对象存储
1. 进入控制台 → 搜索 "COS"
2. 创建存储桶
   - 名称：`family-meal-planner`
   - 访问权限：**公有读私有写**
   - 地域：选择离你最近的

### 步骤3：上传文件
1. 进入存储桶
2. 点击 "上传文件"
3. 将 `dist` 文件夹内的所有文件上传

### 步骤4：开启静态网站
1. 在存储桶左侧菜单找到 **"基础配置"** → **"静态网站"**
2. 点击 "编辑"
3. 启用静态网站
4. 索引文档：`index.html`
5. 保存

### 步骤5：访问应用
你会获得一个访问域名，例如：
```
http://family-meal-planner-123456.cos.ap-beijing.myqcloud.com
```

## 🇨🇳 方案三：阿里云 OSS 静态托管

### 步骤1：注册阿里云
访问 https://www.aliyun.com 注册账号

### 步骤2：创建 OSS Bucket
1. 进入控制台 → 搜索 "OSS"
2. 创建 Bucket
   - 名称：`family-meal-planner`
   - 区域：选择就近区域
   - 读写权限：**公共读**

### 步骤3：上传文件
1. 进入 Bucket
2. 点击 "文件管理" → "上传文件"
3. 上传 `dist` 文件夹内所有文件

### 步骤4：配置静态页面
1. 左侧菜单 → "基础设置" → "静态页面"
2. 默认首页：`index.html`
3. 保存

### 步骤5：访问应用
获得访问域名：
```
http://family-meal-planner.oss-cn-hangzhou.aliyuncs.com
```

## 🌐 方案四：局域网访问（最快，无需部署）

如果只是想临时在手机上使用：

1. **确保手机和电脑在同一WiFi**
2. **查看服务器地址**：
   - 运行 `npm run dev` 后显示的 Network 地址
   - 例如：`http://192.168.192.122:5173/`

3. **手机浏览器输入该地址**

4. **添加到主屏幕**：
   - iPhone: Safari → 分享 → "添加到主屏幕"
   - Android: Chrome → 菜单 → "添加到主屏幕"

> ⚠️ 注意：此方法需要电脑保持开机状态

## 📱 手机端使用提示

### 添加到主屏幕后
- 应用会像原生App一样显示在主屏幕
- 支持离线使用（PWA功能）
- 全屏显示，无浏览器地址栏

### 数据保存
- 所有数据保存在浏览器本地存储中
- 清除浏览器缓存会导致数据丢失
- 建议定期备份重要菜谱

## 🔧 常见问题

### Q: Gitee Pages 访问慢？
A: Gitee Pages 在国内访问速度较快，如果仍然慢可以尝试 CDN 加速。

### Q: 如何更新已部署的应用？
A: 重新构建后上传新的 `dist` 文件夹内容到对应平台即可。

### Q: 数据会同步吗？
A: 当前版本数据保存在本地，不同设备间不会同步。如需多设备同步，建议使用云端数据库方案。
